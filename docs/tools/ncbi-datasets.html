MCP Tools

N NCBI Datasets

Genome and gene summaries, downloads, and batch queries via the NCBI Datasets CLI. Requires the ncbi-datasets CLI to be installed.

Tool Details

datasets_summary_genome

Assembly Summaries

Get summary information for genome assemblies by accession, taxon, or bioproject. Returns metadata including assembly stats, organism info, and annotation details.

Parameters

accession Genome assembly accession (e.g., GCF_000001405.40)
taxon Taxon name or NCBI Taxonomy ID (e.g., 'human' or '9606')
limit Maximum number of results (default: 10)

Example 1: Find human genome assemblies

Natural language: "Show me human genome assemblies"
{
  "taxon": "human",
  "limit": 5
}

Example 2: Get specific assembly by accession

Natural language: "Get details for the GRCh38 human reference assembly"
{
  "accession": "GCF_000001405.40"
}

Example 3: Find assemblies by organism

Natural language: "List all mouse genome assemblies available"
{
  "taxon": "mus musculus",
  "limit": 10
}

datasets_summary_gene

Gene Summaries

Get summary information for genes by gene ID, symbol, or taxon. Returns gene metadata, genomic coordinates, and annotation details.

Parameters

symbol Gene symbol (e.g., 'BRCA1')
gene_id NCBI Gene ID (e.g., '672')
taxon Taxon name or ID to filter results
limit Maximum results (default: 10)

Example 1: Lookup gene by symbol

Natural language: "What is the gene summary for TP53?"
{
  "symbol": "TP53"
}

Example 2: Get gene by NCBI Gene ID

Natural language: "Show me gene information for NCBI Gene ID 672"
{
  "gene_id": "672"
}

Example 3: Find genes in a species

Natural language: "List BRCA genes in human"
{
  "symbol": "BRCA*",
  "taxon": "human",
  "limit": 10
}

datasets_download_genome

Download Assemblies

Download genome assembly data including sequences, annotations, and metadata. Use --dry-run to preview what would be downloaded.

Parameters

accession Genome assembly accession
taxon Taxon name or ID
include Data types: genome, rna, protein, cds, gff3, gtf, gbff, seq-report
dry_run Show what would be downloaded (default: false)

Example 1: Dry run download

Natural language: "What files would be downloaded for human GRCh38?"
{
  "accession": "GCF_000001405.40",
  "dry_run": true
}

Example 2: Download genome and annotation

Natural language: "Download the human genome with GFF annotations"
{
  "accession": "GCF_000001405.40",
  "include": ["genome", "gff3"],
  "filename": "human_grch38.zip"
}

Example 3: Download by taxon

Natural language: "Download latest mouse genome assembly"
{
  "taxon": "mouse",
  "include": ["genome", "gtf"],
  "dry_run": true
}

datasets_download_gene

Download Gene Data

Download gene data including sequences and annotations for specific genes.

Parameters

symbol Gene symbol or comma-separated list
gene_id NCBI Gene ID or comma-separated list
taxon Taxon name or ID to filter
include Data types: gene, rna, protein, cds

Example 1: Download BRCA1 gene

Natural language: "Download the BRCA1 gene data including protein"
{
  "symbol": "BRCA1",
  "taxon": "human",
  "include": ["gene", "protein"],
  "dry_run": true
}

Example 2: Batch download multiple genes

Natural language: "Download data for TP53, BRCA1, and BRCA2"
{
  "symbol": "TP53,BRCA1,BRCA2",
  "taxon": "human",
  "include": ["gene", "rna", "protein"],
  "dry_run": true
}

Example 3: Download by Gene ID

Natural language: "Download gene data for NCBI Gene ID 7157"
{
  "gene_id": "7157",
  "include": ["gene", "protein", "cds"],
  "dry_run": true
}

batch_gene_summary

Multi-Gene Queries

Query multiple genes at once by gene IDs or symbols. Returns consolidated summary results for all genes in a single call.

Parameters

symbols Comma-separated gene symbols (e.g., 'BRCA1,BRCA2,TP53')
gene_ids Comma-separated NCBI Gene IDs
taxon Taxon name or ID to filter

Example 1: Query multiple oncogenes

Natural language: "Get summaries for TP53, KRAS, and EGFR"
{
  "symbols": "TP53,KRAS,EGFR"
}

Example 2: Query by Gene IDs

Natural language: "Get information for genes 7157, 672, 1956"
{
  "gene_ids": "7157,672,1956"
}

Example 3: Breast cancer gene panel

Natural language: "Summarize these breast cancer genes: BRCA1, BRCA2, PALB2, CHEK2"
{
  "symbols": "BRCA1,BRCA2,PALB2,CHEK2",
  "taxon": "human"
}
