E Ensembl

Gene lookup, sequences, variants, homologs, regulatory features, and coordinate liftover from the Ensembl REST API.

Tool Details

ensembl_lookup_gene

Gene Lookup

Look up a gene via Ensembl by stable ID or by symbol and species. Returns genomic coordinates, biotype, description, and cross-references.

Parameters

id Ensembl stable ID (e.g., 'ENSG00000141510')
symbol Gene symbol (e.g., 'TP53'). Requires 'species'.
species Species name (e.g., 'homo_sapiens')

Example 1: Lookup by Ensembl ID

Natural language: "Get gene information for ENSG00000141510"
{ "id": "ENSG00000141510" }

Example 2: Lookup by symbol

Natural language: "Find BRCA1 gene in human"
{
  "symbol": "BRCA1",
  "species": "homo_sapiens"
}

Example 3: Lookup gene in other species

Natural language: "Find TP53 in mouse"
{
  "symbol": "Tp53",
  "species": "mus_musculus"
}

ensembl_get_sequence

Retrieve Sequences

Retrieve nucleotide or protein sequences from Ensembl by stable ID. Supports genomic, cDNA, CDS, and protein sequence types.

Parameters

id Ensembl stable ID (gene, transcript, or protein)
seq_type 'genomic', 'cdna', 'cds', or 'protein' (default: cdna)
format 'json' or 'fasta' (default: json)

Example 1: Get cDNA sequence

Natural language: "Get the cDNA sequence for TP53"
{
  "id": "ENSG00000141510",
  "seq_type": "cdna"
}

Example 2: Get protein sequence

Natural language: "Get the protein sequence for BRCA1"
{
  "id": "ENSG00000012048",
  "seq_type": "protein"
}

Example 3: Get genomic sequence

Natural language: "Get genomic sequence for EGFR"
{
  "id": "ENSG00000146648",
  "seq_type": "genomic"
}

ensembl_get_variants

Regional Variants

Get known genetic variants in a genomic region from Ensembl. Returns variant IDs, alleles, consequences, and clinical significance.

Parameters

region Genomic region as chr:start-end (e.g., '7:140453136-140624564')
species Species name (default: homo_sapiens)
limit Maximum variants (default: 100)

Example 1: Get variants in BRCA1 region

Natural language: "Show variants in the BRCA1 gene region"
{ "region": "17:43044295-43170245" }

Example 2: Variant scan of a region

Natural language: "Find variants in a 10kb region on chromosome 7"
{
  "region": "7:140453136-140463136",
  "limit": 50
}

Example 3: TP53 variant region

Natural language: "Show variants in TP53 genomic region"
{
  "region": "17:7661779-7687550",
  "limit": 200
}

ensembl_get_homologs

Orthologs and Paralogs

Find homologous genes (orthologs/paralogs) for an Ensembl gene ID. Useful for cross-species gene comparisons.

Parameters

id Ensembl gene ID (e.g., 'ENSG00000141510')
homology_type 'orthologues', 'paralogues', or 'all' (default: all)
target_species Target species for orthologs (e.g., 'mus_musculus')

Example 1: Find all orthologs

Natural language: "Find orthologs of human TP53 in other species"
{
  "id": "ENSG00000141510",
  "homology_type": "orthologues"
}

Example 2: Mouse ortholog only

Natural language: "What is the mouse ortholog of human BRCA1?"
{
  "id": "ENSG00000012048",
  "target_species": "mus_musculus"
}

Example 3: Find paralogs

Natural language: "Find paralogs of the KRAS gene"
{
  "id": "ENSG00000133703",
  "homology_type": "paralogues"
}

ensembl_get_regulation

Regulatory Features

Get regulatory features (promoters, enhancers, CTCF binding sites, open chromatin, TF binding sites) in a genomic region.

Parameters

region Genomic region as chr:start-end
species Species name (default: homo_sapiens)

Example 1: Get regulatory elements near TP53

Natural language: "Show regulatory features near TP53"
{ "region": "17:7661779-7687550" }

Example 2: Find enhancers in a region

Natural language: "What regulatory elements are in this region?"
{ "region": "7:140453136-140624564" }

Example 3: Promoter regions

Natural language: "Find promoter and enhancer elements"
{ "region": "1:1-1000000" }

liftover_coordinates

Coordinate Liftover

Convert genomic coordinates between genome assemblies (e.g., GRCh37/hg19 to GRCh38/hg38 or vice versa) using Ensembl.

Parameters

region Genomic region as chr:start-end
source_assembly 'GRCh37' or 'GRCh38' (default: GRCh37)
target_assembly 'GRCh37' or 'GRCh38' (default: GRCh38)
species Species name (default: human)

Example 1: Convert hg19 to hg38

Natural language: "Convert this position from hg19 to hg38"
{
  "region": "17:43044295-43170245",
  "source_assembly": "GRCh37",
  "target_assembly": "GRCh38"
}

Example 2: Convert hg38 to hg19

Natural language: "Convert this hg38 coordinate to hg19"
{
  "region": "chr7:140453136-140624564",
  "source_assembly": "GRCh38",
  "target_assembly": "GRCh37"
}

Example 3: Single position conversion

Natural language: "What is the hg38 coordinate for position 7:140500000?"
{
  "region": "7:140500000-140500000",
  "source_assembly": "GRCh37",
  "target_assembly": "GRCh38"
}
