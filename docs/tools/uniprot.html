U UniProt & Protein

Protein search, function, domains, features, interactions, and domain architecture from UniProt, InterPro, and STRING.

Tool Details

uniprot_search

Protein Search

Search UniProt for proteins using Lucene query syntax. Supports filtering by organism, gene name, protein name, GO terms, and more.

Parameters

query UniProt search query in Lucene syntax
limit Maximum results (default: 10)
reviewed Only return Swiss-Prot (reviewed) entries

Example 1: Search for a protein

Natural language: "Search for TP53 protein in human"
{
  "query": "TP53 AND organism_id:9606",
  "limit": 5
}

Example 2: Search kinase family

Natural language: "Find human protein kinases"
{
  "query": "kinase AND organism_id:9606 AND reviewed:true",
  "limit": 10
}

Example 3: Search by function

Natural language: "Find transcription factors in human"
{
  "query": "transcription factor AND organism_id:9606",
  "limit": 10
}

uniprot_get_protein

Protein Function

Get detailed protein information from UniProt by accession. Returns function, subcellular location, GO terms, and key annotations.

Parameters

accession UniProt accession (e.g., 'P04637' for human TP53)

Example 1: Get TP53 protein info

Natural language: "Get detailed information for TP53 protein"
{ "accession": "P04637" }

Example 2: Get BRCA1 details

Natural language: "Get protein function for BRCA1"
{ "accession": "P38398" }

Example 3: Get kinase info

Natural language: "Get details for EGFR protein"
{ "accession": "P00533" }

uniprot_get_features

Protein Features

Get protein sequence features (domains, active sites, modifications, variants) from UniProt by accession.

Parameters

accession UniProt accession
feature_types Filter by types: 'Domain', 'Active site', 'Modified residue', etc.

Example 1: Get all features

Natural language: "Show all features for TP53"
{ "accession": "P04637" }

Example 2: Get only domains

Natural language: "What domains does BRCA1 have?"
{
  "accession": "P38398",
  "feature_types": ["Domain"]
}

Example 3: Get modified residues

Natural language: "Show phosphorylation sites in EGFR"
{
  "accession": "P00533",
  "feature_types": ["Modified residue"]
}

interpro_get_domains

Domain Architecture

Get protein domain and family annotations from InterPro by UniProt accession. Returns domain architecture including Pfam, PROSITE, CDD, and other member database matches.

Parameters

accession UniProt accession

Example 1: Get domain architecture

Natural language: "What domains does TP53 have?"
{ "accession": "P04637" }

Example 2: Get BRCA1 domains

Natural language: "Show domain structure of BRCA1"
{ "accession": "P38398" }

Example 3: Kinase domains

Natural language: "What are the kinase domains in EGFR?"
{ "accession": "P00533" }

string_get_interactions

Protein-Protein Interactions

Get protein-protein interaction partners from STRING database. Returns interaction scores, evidence channels, and network neighbors.

Parameters

identifiers Protein name(s), separated by newline or comma
limit Max partners per protein (default: 10)
required_score Minimum combined score (0-1000, default: 400)
network_type 'functional' or 'physical' (default: functional)
species NCBI taxonomy ID (default: 9606 for human)

Example 1: Find TP53 interactors

Natural language: "What proteins interact with TP53?"
{ "identifiers": "TP53" }

Example 2: Protein network

Natural language: "Show interaction network for BRCA1, BRCA2, and PALB2"
{
  "identifiers": "BRCA1,BRCA2,PALB2",
  "limit": 15
}

Example 3: High confidence interactions

Natural language: "Show only high-confidence interactions with EGFR"
{
  "identifiers": "EGFR",
  "required_score": 700,
  "limit": 20
}
