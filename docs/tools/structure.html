S Structure & Function

3D structures, predicted models, pathways, expression, and functional enrichment from PDB, AlphaFold, KEGG, Reactome, GTEx, and g:Profiler.

Tool Details

pdb_search & pdb_get_structure

Protein Data Bank

Search RCSB PDB for 3D structures by free-text query, or get detailed metadata for a specific PDB ID.

Parameters (pdb_search)

query Search text (e.g., 'TP53', 'insulin receptor')
limit Maximum results (default: 10)

Parameters (pdb_get_structure)

pdb_id 4-character PDB ID (e.g., '1TUP', '4HHB')

Example 1: Search for TP53 structures

Natural language: "Find all TP53 structures in PDB"
{
  "query": "TP53",
  "limit": 10
}

Example 2: Get structure details

Natural language: "Get details for PDB 1TUP"
{ "pdb_id": "1TUP" }

Example 3: Search kinase structures

Natural language: "Find kinase structures"
{
  "query": "kinase",
  "limit": 15
}

AlphaFold Predictions

Get AlphaFold predicted 3D structure information for a protein by UniProt accession.

Parameters

accession UniProt accession (e.g., 'P04637' for TP53)

Example 1: Get TP53 AlphaFold

Natural language: "Get AlphaFold prediction for TP53"
{ "accession": "P04637" }

Example 2: Get BRCA1 prediction

Natural language: "What is the AlphaFold structure for BRCA1?"
{ "accession": "P38398" }

Example 3: Get kinase structure

Natural language: "Get AlphaFold model for EGFR"
{ "accession": "P00533" }

kegg_get_pathway

KEGG Pathways

Search and retrieve KEGG pathway information. Returns pathway names, descriptions, genes, and linked entries.

Parameters

query Keyword search or KEGG pathway ID (e.g., 'hsa04210')
organism KEGG organism code (e.g., 'hsa' for human)

Example 1: Search for apoptosis pathway

Natural language: "Find apoptosis pathway in KEGG"
{ "query": "apoptosis" }

Example 2: Get specific pathway

Natural language: "Get the p53 signaling pathway"
{ "query": "hsa04215" }

Example 3: Cell cycle pathway

Natural language: "Show the cell cycle pathway"
{
  "query": "cell cycle",
  "organism": "hsa"
}

reactome_get_pathway

Reactome Pathways

Get detailed pathway information from Reactome — the curated knowledgebase of biological pathways.

Parameters

pathway_id Reactome stable ID (e.g., 'R-HSA-1640170')
query Keyword search for pathways
include_participants Include participating molecules (default: false)
limit Maximum search results (default: 10)

Example 1: Search pathways

Natural language: "Find p53 pathway in Reactome"
{
  "query": "p53",
  "limit": 5
}

Example 2: Get pathway with genes

Natural language: "Get apoptosis pathway with all genes"
{
  "pathway_id": "R-HSA-109581",
  "include_participants": true
}

Example 3: DNA repair pathway

Natural language: "Find DNA repair pathways"
{
  "query": "DNA repair",
  "limit": 10
}

gtex_get_expression

GTEx Expression

Get tissue-specific gene expression data from GTEx. Returns median TPM expression values across ~54 human tissues.

Parameters

gene Gene symbol (e.g., 'TP53') or Ensembl gene ID
dataset 'gtex_v8' or 'gtex_v10' (default: gtex_v8)

Example 1: TP53 expression

Natural language: "What is TP53 expression across tissues?"
{ "gene": "TP53" }

Example 2: Expression by Ensembl ID

Natural language: "Get expression for ENSG00000141510"
{ "gene": "ENSG00000141510" }

Example 3: Tissue-specific expression

Natural language: "Show BRCA1 expression in different tissues"
{ "gene": "BRCA1" }

gene_ontology_enrich

GO Enrichment Analysis

Perform Gene Ontology (GO) enrichment analysis on a gene list using g:Profiler.

Parameters

genes Comma-separated gene symbols
organism g:Profiler organism (default: 'hsapiens')
sources Data sources (default: 'GO:BP,GO:MF,GO:CC')
threshold Significance threshold (default: 0.05)
limit Maximum terms (default: 25)

Example 1: Basic enrichment

Natural language: "What pathways are enriched in TP53, BRCA1, EGFR?"
{ "genes": "TP53,BRCA1,EGFR" }

Example 2: Include KEGG pathways

Natural language: "Find GO and KEGG enrichment for these genes"
{
  "genes": "TP53,KRAS,MYC,BRCA1",
  "sources": "GO:BP,GO:MF,GO:CC,KEGG"
}

Example 3: Mouse gene list

Natural language: "Enrichment analysis for mouse genes"
{
  "genes": "Trp53,Brca1,Egfr,Kras",
  "organism": "mmusculus",
  "limit": 20
}
