MCP Tools

C Clinical & Variants

Clinical variant interpretation, population frequencies, phenotype associations, and disease mapping from ClinVar, gnomAD, HPO, OMIM, COSMIC, and Disease Ontology.

Tool Details

clinvar_search

Clinical Significance

Search NCBI ClinVar for clinical variant interpretations. Query by gene, variant, disease, or condition.

Parameters

query ClinVar search query (e.g., 'BRCA1', 'BRCA1 AND pathogenic')
limit Maximum results (default: 20)

Example 1: Search for pathogenic variants

Natural language: "Find pathogenic BRCA1 variants in ClinVar"
{
  "query": "BRCA1 AND pathogenic",
  "limit": 10
}

Example 2: Search by variant notation

Natural language: "Find the c.5266dupC variant in BRCA1"
{ "query": "NM_007294.4:c.5266dupC" }

Example 3: Search by disease

Natural language: "Find variants associated with breast cancer"
{
  "query": "breast cancer",
  "limit": 15
}

gnomad_get_variant

Population Frequencies

Get population allele frequencies for a genetic variant from gnomAD.

Parameters

variant_id Variant ID (e.g., '1-55516888-G-A') or rsID (e.g., 'rs11549407')
dataset gnomAD dataset version (default: 'gnomad_r4')

Example 1: Query by rsID

Natural language: "What is the population frequency of rs11549407?"
{ "variant_id": "rs11549407" }

Example 2: Query by position

Natural language: "Get frequency for 1-55516888-G-A"
{ "variant_id": "1-55516888-G-A" }

Example 3: BRCA1 V600E frequency

Natural language: "What is the frequency of the BRAF V600E variant?"
{ "variant_id": "7:140453136-A-G" }

hpo_search

Phenotype Ontology

Search the Human Phenotype Ontology (HPO) for phenotype terms, or get genes/diseases associated with a phenotype.

Parameters

query Keyword, HPO term ID (e.g., 'HP:0001250'), or gene symbol
category 'search', 'genes', 'diseases', or 'term' (default: search)
limit Maximum results (default: 20)

Example 1: Search for phenotypes

Natural language: "Search for seizure-related phenotypes"
{
  "query": "seizure",
  "limit": 10
}

Example 2: Get genes for a phenotype

Natural language: "Which genes are associated with HP:0001250?"
{
  "query": "HP:0001250",
  "category": "genes"
}

Example 3: Find diseases for phenotype

Natural language: "What diseases have intellectual disability?"
{
  "query": "intellectual disability",
  "category": "diseases"
}

omim_search

Mendelian Disease

Search OMIM for genetic disease associations. Returns MIM numbers, titles, and entry types.

Parameters

query Gene symbol, disease name, or MIM number
gene_id NCBI Gene ID to get linked diseases
limit Maximum results (default: 20)

Example 1: Search for disease

Natural language: "Search for Marfan syndrome in OMIM"
{ "query": "Marfan syndrome" }

Example 2: Get diseases for a gene

Natural language: "What diseases are linked to TP53?"
{ "gene_id": "7157" }

Example 3: Search by MIM number

Natural language: "Get OMIM entry 191170"
{ "query": "191170" }

cosmic_search

Somatic Mutations

Search the COSMIC database for somatic mutations in cancer.

Parameters

query Gene name, mutation (e.g., 'V600E'), or keyword
gene Filter results to a specific gene
limit Maximum mutations (default: 20, max: 500)

Example 1: Search for BRAF V600E

Natural language: "Find COSMIC mutations for BRAF V600E"
{
  "query": "BRAF V600E",
  "limit": 10
}

Example 2: All TP53 mutations

Natural language: "Show all TP53 somatic mutations"
{
  "gene": "TP53",
  "limit": 20
}

Example 3: KRAS mutations in lung cancer

Natural language: "Find KRAS mutations in lung cancer"
{
  "query": "KRAS lung",
  "limit": 15
}

disease_ontology_search

Disease Terms

Search the Disease Ontology (DO) for standardized disease terms, definitions, and cross-references.

Parameters

query Disease name, DOID (e.g., 'DOID:1612'), or keyword
limit Maximum results (default: 10)

Example 1: Search for disease

Natural language: "Search for breast cancer in Disease Ontology"
{ "query": "breast cancer" }

Example 2: Get DOID details

Natural language: "Get information for DOID:1612"
{ "query": "DOID:1612" }

Example 3: Find related diseases

Natural language: "Find diseases similar to Alzheimer's"
{
  "query": "Alzheimer",
  "limit": 10
}
