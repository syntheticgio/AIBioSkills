MCP Tools

B BioPython Analysis

Local sequence analysis — alignment, statistics, translation, primer design, and FASTA parsing. All tools run locally using BioPython with no external API dependencies.

Tool Details

sequence_align

Pairwise Sequence Alignment

Perform pairwise sequence alignment using BioPython. Returns aligned sequences, identity percentage, gap count, and alignment score. Supports global and local alignment modes.

Parameters

sequence1 First sequence (raw or FASTA format)
sequence2 Second sequence (raw or FASTA format)
mode 'global' or 'local' (default: global)
sequence_type 'protein' or 'nucleotide' (default: auto-detect)

Example 1: Global protein alignment

Natural language: "Align these two protein sequences globally"
{
  "sequence1": "MVLSPADKTNVKAAWGKVGAHAGEYGAEALERMFLSFPTTKTYFPHFDLSH",
  "sequence2": "MVLSGEDKSNIKAAWGKIGGHGAEYGAEALERMFASFPTTKTYFPHFDVSH",
  "sequence_type": "protein",
  "mode": "global"
}

Example 2: Local alignment for finding motifs

Natural language: "Find conserved regions between these sequences"
{
  "sequence1": "ATGCGATCGATCGATCGATCG",
  "sequence2": "GGATCGATCGATCGAAAA",
  "sequence_type": "nucleotide",
  "mode": "local"
}

Example 3: DNA sequence comparison

Natural language: "Compare these two DNA promoter sequences"
{
  "sequence1": "TATAAA",
  "sequence2": "TATATA",
  "sequence_type": "nucleotide"
}

sequence_stats

Sequence Statistics

Compute statistics for a nucleotide or protein sequence. Returns length, GC content (nucleotide), codon usage (nucleotide), amino acid composition (protein), and molecular weight (protein).

Parameters

sequence Raw sequence or path to FASTA file

Example 2: GC content analysis

Natural language: "Calculate GC content for this DNA sequence"

sequence_translate

Six-Frame Translation

Translate a nucleotide sequence to protein in all 6 reading frames (3 forward + 3 reverse complement). Returns amino acid sequences with stop codons marked as '*'.

Parameters

sequence Nucleotide sequence (raw or FASTA)
frames 'all', 'forward', 'reverse', or specific frame like '+1', '-2'
table NCBI genetic code table (default: 1 = Standard)

Example 1: Translate in all frames

Natural language: "Translate this sequence in all 6 reading frames"
{
  "sequence": "ATGCGTACGTTAGCTAGCTAG",
  "frames": "all"
}

Example 2: Forward frames only

Natural language: "Show forward strand translations"
{
  "sequence": "ATGATAGCTGATCGATCGATCGAT",
  "frames": "forward"
}

Example 3: Mitochondrial translation

Natural language: "Translate using vertebrate mitochondrial code"
{
  "sequence": "ATGCGTAGCTAGCTAGCTAGCTA",
  "frames": "all",
  "table": 2
}

parse_fasta

FASTA Parsing

Parse a FASTA file and return sequence IDs, descriptions, lengths, and optionally full sequences. Supports filtering by regex pattern on IDs/descriptions.

Parameters

file_path Path to the FASTA file
filter_pattern Regex to filter sequence IDs/descriptions
include_sequences Include full sequences (default: false)
limit Maximum sequences to return (default: 100)

Example 1: List all sequences in file

Natural language: "What sequences are in this FASTA file?"
{
  "file_path": "/path/to/sequences.fasta",
  "include_sequences": false
}

Example 2: Filter by gene name

Natural language: "Extract only TP53 sequences from this file"
{
  "file_path": "/path/to/sequences.fasta",
  "filter_pattern": "TP53",
  "include_sequences": true
}

Example 3: Get first 10 sequences

Natural language: "Show me the first 10 sequences"
{
  "file_path": "/path/to/sequences.fasta",
  "limit": 10,
  "include_sequences": true
}

primer_design

PCR Primer Design

Design PCR primers for a target DNA sequence using Primer3 via BioPython. Returns primer pairs with Tm, GC%, length, and product size.

Parameters

sequence Target nucleotide sequence (min 100bp)
target_length Amplicon size (default: 200bp)
primer_min_tm Min Tm (default: 57°C)
primer_max_tm Max Tm (default: 63°C)
num_primers Number of primer pairs (default: 5)

Example 1: Design standard PCR primers

Natural language: "Design primers for a 200bp amplicon"
{
  "sequence": "ATGCGTACGTTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGC",
  "target_length": 200
}

Example 2: High-Tm primers

Natural language: "Design primers with higher Tm for GC-rich template"
{
  "sequence": "GCGCGCGCGCGATCGATCGATCGATCGATCGATCGATCGATCGATCGATCGATCGATCGATCGAT",
  "primer_min_tm": 60,
  "primer_max_tm": 68,
  "target_length": 150
}

Example 3: Long amplicon primers

Natural language: "Design primers for a 500bp fragment"
{
  "sequence": "ATGCGTACGTTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGCTAGC",
  "target_length": 500,
  "num_primers": 3
}
