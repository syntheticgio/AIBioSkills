L Literature & Papers

Search PubMed, fetch full-text papers, and get citation data from Semantic Scholar.

Tool Details

pubmed_search

PubMed Search

Search PubMed for biomedical literature. Returns article titles, authors, journals, dates, abstracts, DOIs, and PMIDs.

Parameters

query PubMed search query (e.g., 'BRCA1 AND PARP inhibitor')
limit Maximum results (default: 10)
sort 'relevance', 'date', or 'author' (default: relevance)

Example 1: Search for BRCA1 papers

Natural language: "Find recent papers on BRCA1 and PARP inhibitors"
{
  "query": "BRCA1 AND PARP inhibitor",
  "limit": 10
}

Example 2: Recent TP53 research

Natural language: "Find recent TP53 cancer therapy papers"
{
  "query": "TP53 cancer therapy 2024",
  "sort": "date",
  "limit": 10
}

Example 3: CRISPR papers

Natural language: "Search for CRISPR gene editing papers"
{
  "query": "CRISPR gene editing",
  "limit": 15
}

paper_fetch

Full-Text Paper Retrieval

Retrieve a biomedical research paper's metadata and full text (when available via PubMed Central or Europe PMC).

Parameters

identifier PMID, DOI, PMC ID, or article title
sections 'methods', 'results', 'discussion', 'introduction', 'all' (default: methods, results)

Example 1: Fetch by PMID

Natural language: "Get the paper with PMID 33057194"
{
  "identifier": "33057194",
  "sections": ["methods", "results"]
}

Example 2: Fetch by DOI

Natural language: "Get the paper with DOI 10.1038/s41586-020-2649-2"
{ "identifier": "10.1038/s41586-020-2649-2" }

Example 3: Get all sections

Natural language: "Get full text of a paper"
{
  "identifier": "PMC7505768",
  "sections": ["all"]
}

semantic_scholar_search

Citations & Impact

Search Semantic Scholar for paper metadata, citation counts, influential citations, TLDRs, and open-access PDF links.

Parameters

query Paper title, 'DOI:10.1038/...' for DOI, 'PMID:...' for PMID, or keyword search
limit Max results for keyword (default: 5)
fields_of_study Filter by field: 'Medicine', 'Biology', etc.

Example 1: Search by title

Natural language: "Find the paper 'Attention is All You Need'"
{ "query": "Attention is All You Need" }

Example 2: Search by DOI

Natural language: "Get citation data for this DOI"
{ "query": "DOI:10.1038/nature14539" }

Example 3: Biology papers

Natural language: "Find highly cited biology papers"
{
  "query": "CRISPR",
  "fields_of_study": "Biology",
  "limit": 10
}
