MCP Tools
50+ Tools Across Every Major Database

Each tool is callable by any MCP-compatible AI assistant. Just describe what you need.

N NCBI Datasets

Genome and gene summaries, downloads, and batch queries via the NCBI Datasets CLI.

  • datasets_summary_genome Assembly summaries
  • datasets_summary_gene Gene summaries
  • datasets_download_genome Download assemblies
  • datasets_download_gene Download gene data
  • batch_gene_summary Multi-gene queries

B BioPython Analysis

Local sequence analysis — alignment, statistics, translation, primer design, and FASTA parsing.

  • sequence_align Pairwise alignment
  • sequence_stats GC%, MW, composition
  • sequence_translate 6-frame translation
  • parse_fasta FASTA parsing & filtering
  • primer_design PCR primer design

E Ensembl

Gene lookup, sequences, variants, homologs, regulatory features, and coordinate liftover.

  • ensembl_lookup_gene Gene lookup by ID/symbol
  • ensembl_get_sequence Retrieve sequences
  • ensembl_get_variants Regional variants
  • ensembl_get_homologs Orthologs/paralogs
  • ensembl_get_regulation Regulatory features
  • liftover_coordinates hg19 ↔ hg38

U UniProt & Protein

Protein search, function, domains, features, interactions, and domain architecture.

  • uniprot_search Protein search
  • uniprot_get_protein Function & GO terms
  • uniprot_get_features Domains & PTMs
  • interpro_get_domains Domain architecture
  • string_get_interactions PPI networks

C Clinical & Variants

Clinical variant interpretation, population frequencies, phenotype associations, and disease mapping.

  • clinvar_search Clinical significance
  • gnomad_get_variant Population frequencies
  • hpo_search Phenotype ontology
  • omim_search Mendelian disease
  • cosmic_search Somatic mutations
  • disease_ontology_search Disease terms

S Structure & Function

3D structures, predicted models, pathways, expression, and functional enrichment.

  • pdb_get_structure / pdb_search PDB
  • alphafold_get_prediction AlphaFold
  • kegg_get_pathway KEGG pathways
  • reactome_get_pathway Reactome
  • gtex_get_expression Tissue expression
  • gene_ontology_enrich GO enrichment

L Literature & Papers

Search PubMed, fetch full-text papers, get citation data and impact metrics.

  • pubmed_search PubMed search
  • paper_fetch Full-text retrieval
  • semantic_scholar_search Citations & impact

G Genomics & Epigenomics

ENCODE experiments, expression atlas, gene links, and pharmacogenomics.

  • encode_get_experiments ENCODE data
  • expression_atlas_search Expression Atlas
  • ncbi_gene_links Gene relationships
  • pharmgkb_search Pharmacogenomics
  • blast_search Sequence similarity

Getting Started
Install in Minutes

No API keys required. Most tools work without any external CLI dependencies.

1

Install the MCP Server

Install from GitHub (PyPI coming soon):

$ pip install git+https://github.com/syntheticgio/datasets-mcp.git

# or for development:
$ git clone https://github.com/syntheticgio/datasets-mcp.git
$ cd datasets-mcp && pip install -e .
2

Configure Your Client

Add to your Claude Code or Claude Desktop config:

{
  "mcpServers": {
    "datasets": {
      "command": "datasets-mcp-server"
    }
  }
}
3

Install Skills (Optional)

For Claude Code slash commands — copy or symlink:

$ mkdir -p ~/.claude/commands
$ cp .claude/commands/*.md ~/.claude/commands/

# or symlink to stay up-to-date:
$ for f in .claude/commands/*.md; do
    ln -sf "$(pwd)/$f" ~/.claude/commands/
  done

Optional: NCBI Datasets CLI

Only needed for datasets_* and batch_gene_summary tools. All other tools (Ensembl, UniProt, BioPython, etc.) work without it.

# Conda (recommended)
$ conda install -c conda-forge ncbi-datasets-cli

# Direct download — Linux (amd64)
$ curl -o datasets 'https://ftp.ncbi.nlm.nih.gov/pub/datasets/command-line/v2/linux-amd64/datasets'
$ chmod +x datasets && sudo mv datasets /usr/local/bin/

# Direct download — macOS
$ curl -o datasets 'https://ftp.ncbi.nlm.nih.gov/pub/datasets/command-line/v2/mac/datasets'
$ chmod +x datasets && sudo mv datasets /usr/local/bin/

# Windows (PowerShell)
> curl -o datasets.exe https://ftp.ncbi.nlm.nih.gov/pub/datasets/command-line/v2/win64/datasets.exe

See the NCBI install guide for ARM64, Linux 32-bit, and other platforms.

Use Cases
What Can You Do?

Ask questions in natural language. The assistant picks the right tools automatically.

Gene Research

"Tell me everything about TP53 — function, domains, structures, variants, interactions, and pathways."

/gene-report TP53

Variant Interpretation

"What is the clinical significance of BRAF V600E? Show population frequencies, domain impact, and phenotype associations."

/clinical-variant-agent BRAF V600E

Drug Target Assessment

"Is EGFR druggable? Show me ligand-bound structures, binding sites, COSMIC hotspots, and disease rationale."

/drug-target-report EGFR

Comparative Genomics

"Compare BRCA1 across human, mouse, zebrafish, and chicken — show me conservation and divergent regions."

/comparative-genomics-agent BRCA1 human mouse zebrafish chicken

Gene List Analysis

"What biological themes connect TP53, BRCA1, EGFR, PTEN, and RB1? Show shared pathways, interactions, and expression."

/gene-list-agent TP53,BRCA1,EGFR,PTEN,RB1

Paper Analysis

"Analyze all statistical methods in PMID 33057194 — check assumptions, appropriateness, and interpretation."

/statistical-methods-agent 33057194
