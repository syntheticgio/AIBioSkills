AlphaFold

AI-Predicted 3D Protein Structures with Confidence Scores

Home › AI-Powered Predictions — AlphaFold uses deep neural networks to predict protein 3D structures from amino acid sequences, providing rapid structure availability for millions of proteins.

Key Features

  • AI-predicted 3D structures
  • pLDDT confidence scores
  • Predicted aligned error (PAE)
  • Genome-scale coverage
  • Multiple organism support
  • Accessible predictions

Use Cases

  • Rapidly obtain protein structures
  • Assess prediction confidence
  • Study multi-chain interactions
  • Design protein variants
  • Map variants to structure
  • Model protein dynamics

Structure Prediction Retrieval

alphafold_get_prediction

Retrieve AI-predicted protein structure from AlphaFold database.

Parameters: uniprot_id (string: UniProt accession), organism (string, optional: "Homo sapiens", "Mus musculus", etc.)

Returns: PDB structure file, pLDDT confidence scores per residue, PAE matrix, download URLs, model version

Input Options:

  • UniProt accession (e.g., P04637 for TP53_HUMAN)
  • Gene symbol with organism specification
  • UniProt entry name

Confidence Scores & Quality Assessment

pLDDT (per-residue confidence)

pLDDT scores range from 0 to 100 and indicate the predicted local distance difference test score, representing confidence in local structure prediction:

  • 90-100: Very high confidence — structure likely correct
  • 70-90: High confidence — backbone well predicted
  • 50-70: Medium confidence — general fold correct but details uncertain
  • < 50: Low confidence — use with caution, likely disordered region

PAE (Predicted Aligned Error)

PAE provides confidence in relative position and orientation of different protein regions. Low PAE between regions indicates reliable inter-domain distances and orientations. Useful for assessing multi-chain and multi-domain predictions.

Multi-Chain Predictions

AlphaFold can predict protein complexes and multi-chain assemblies, important for understanding protein-protein interactions and functional assemblies.

Comparison with Experimental Structures

When to Use AlphaFold

  • No experimental structure available (PDB)
  • Quick structure assessment needed
  • Genome-wide structural analysis
  • Protein engineering and design
  • Variant impact assessment

Complementarity with PDB

AlphaFold predictions complement experimental structures from PDB. For well-studied proteins with high-resolution experimental structures, PDB remains the gold standard. AlphaFold extends coverage to proteins without experimental structures.

Integration with Datasets MCP

AlphaFold predictions enhance structural analysis across datasets:

  • With UniProt: Map protein features to AlphaFold-predicted structure
  • With PDB: Compare experimental vs. predicted structures
  • With InterPro: Validate domain predictions in structural context
  • With Ensembl: Assess impact of variants on predicted structure
