STRING

Protein-Protein Interactions, Functional Enrichment, and Interaction Networks

Home › Databases › STRING

Overview

STRING is a database of known and predicted protein-protein interactions (PPIs) for over 19,000 organisms. It integrates experimental data, computational predictions, and orthologous information to provide a comprehensive view of protein interaction networks with confidence scores for each interaction.

High-Quality Interaction Data — STRING combines experimental biochemical evidence with computational predictions, providing confidence scores ranging from 0.15 (low) to 1.0 (highest confidence).

Key Features

  • Protein-protein interactions
  • Confidence scores for interactions
  • Evidence sources (experimental, database, text-mining)
  • Functional enrichment analysis
  • Network visualization
  • Multi-organism coverage

Use Cases

  • Find interaction partners for a protein
  • Build protein interaction networks
  • Identify functional modules
  • Infer protein function
  • Analyze disease-related networks
  • Predict novel drug targets

Interaction Search

string_get_interactions

Retrieve protein interaction partners and network data.

Parameters: protein (string: gene symbol or protein name), organism (string, optional: e.g., "Homo sapiens" or NCBI taxon ID)

Returns: Interaction partners, interaction scores, evidence types, enriched pathways, functional annotations

Evidence Types:

  • experimental — Physical interaction data from biochemical assays
  • database — Interactions from curated interaction databases
  • textmining — Co-occurrence in scientific literature
  • coexpression — Gene expression correlation
  • homology — Conserved interactions from orthologs

Example Queries

Find interaction partners

"What proteins interact with the human TP53 protein?"

Confidence Scores Explained

Combined Score Interpretation

STRING provides a combined confidence score for each interaction, integrating multiple evidence sources:

  • 0.90-1.00: Highest confidence interactions (multiple evidence types)
  • 0.70-0.89: High confidence (strong experimental or computational support)
  • 0.40-0.69: Medium confidence (some evidence support)
  • 0.15-0.39: Low confidence (weak or single evidence type)

Evidence Weighting

Different evidence types have different weights:

  • Experimental: Direct biochemical evidence (highest weight)
  • Database: Manually curated interactions
  • Co-expression: Gene expression correlation
  • Text-mining: Literature co-occurrence
  • Homology: Orthologous interactions
