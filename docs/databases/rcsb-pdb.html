Overview

The Research Collaboratory for Structural Bioinformatics (RCSB) maintains the Protein Data Bank (PDB), a global repository of 3D structures of proteins, nucleic acids, and other biological macromolecules. PDB contains over 200,000 experimentally determined structures from X-ray crystallography, NMR spectroscopy, and cryo-electron microscopy (cryo-EM).

Experimental Diversity — PDB includes structures solved by multiple techniques with varying resolutions, providing insights into protein dynamics, drug binding, and molecular mechanisms.

Key Features

  • 3D atomic coordinates
  • Multiple experimental methods
  • High-resolution structure data
  • Ligand and binding site information
  • Biological assembly models
  • Structure quality metrics

Use Cases

  • Visualize protein 3D structure
  • Study drug-protein interactions
  • Analyze enzyme mechanisms
  • Compare homologous structures
  • Map variants to structural regions
  • Design novel therapeutics

Structure Search

pdb_search

Search PDB structures by protein name, PDB ID, or experimental method.

Parameters: query (string: protein name or PDB ID), method (string, optional: x-ray, nmr, cryo-em, all)

Returns: PDB ID, title, organism, method, resolution, release date, ligands, UniProt links

Experimental Methods:

  • X-ray Crystallography: Highest resolution, most structures (~85%)
  • NMR Spectroscopy: Solution structures, dynamics information (~5%)
  • Cryo-EM: Large complexes, membrane proteins (~8%)
  • Others: Small-angle scattering, electron microscopy, etc. (~2%)

Example Queries

Find structures of a protein

"What 3D structures of human hemoglobin are available in PDB?"

Tool: pdb_search with query="hemoglobin"

Search by experimental method

"Find all cryo-EM structures of the ribosome."

Tool: pdb_search with query="ribosome" and method="cryo-em"

Retrieve a specific structure

"Get structure information for PDB entry 1A8O (BRCA1-DNA binding domain)."

Tool: pdb_search with query="1A8O"

Find drug-target complexes

"What structures show how kinase inhibitors bind to the EGFR protein?"

Tool: pdb_search with query="EGFR AND inhibitor"

Structure Quality & Resolution

Resolution Guidelines

  • < 2.0 Å: Excellent — Atomic positions well-determined, side-chain orientations clear
  • 2.0-2.5 Å: Good — Protein fold and secondary structure clearly visible
  • 2.5-3.0 Å: Fair — Main chain visible, some side-chain ambiguity
  • > 3.0 Å: Low — Useful for overall fold but limited detail
  • NMR/Cryo-EM: Reported as ensemble size or local resolution maps

Quality Metrics

  • R-factor/R-free: Agreement between model and experimental data
  • RMSD: Root-mean-square deviation for homologous structures
  • Ramachandran Plot: Phi/psi angle distribution validation
  • Molprobity Score: Overall structure quality assessment

File Formats & Downloads

Standard Formats

  • PDB Format: Classic text format with coordinate records
  • mmCIF: Crystallographic Information File (modern standard)
  • Ligand SDF: Small molecule structures in 3D
  • FASTA: Sequence files for alignment and analysis

Visualization Tools

Access structure visualization through the RCSB PDB website interface, or download coordinates for analysis with tools like PyMOL, Chimera, or Jmol.

Integration with Datasets MCP

PDB structures integrate with other bioinformatics data:

  • With UniProt: Map protein sequences to 3D structure regions
  • With Ensembl: Connect genomic variants to protein structure positions
  • With ClinVar: Visualize disease-associated mutations in 3D context
  • With STRING: View protein-protein interactions in structural context
