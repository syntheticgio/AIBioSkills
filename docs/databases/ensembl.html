Ensembl

Genomic Coordinates, Sequences, Variants, Orthologs, and Regulatory Features

Home › Databases › Ensembl

Overview

Ensembl is the primary European bioinformatics resource for genomic annotation, maintained by the EBI (European Bioinformatics Institute). It provides automatically annotated genomes for 100+ species, with comprehensive information on genes, variants, regulatory features, and evolutionary relationships.

Coverage: Ensembl covers human, model organisms (mouse, fruit fly, yeast, nematode, zebrafish), and over 90 additional species with consistent annotation pipelines.

Key Features

  • Gene lookup and sequences
  • Genomic coordinate conversion (liftover)
  • Regional variant information
  • Ortholog and paralog identification
  • Regulatory element annotation
  • Protein domain mapping

Use Cases

  • Get gene coordinates on a chromosome
  • Find all variants in a genomic region
  • Identify orthologs in other species
  • Map variants to protein domains
  • Convert coordinates between genome builds
  • Find regulatory elements near a gene

Available Data Types

Data Type Description Accessible Via
Gene Information Gene symbols, IDs, descriptions, chromosomal location, exon structure ensembl_lookup_gene
Sequences Genomic, CDS, and protein sequences in FASTA format ensembl_get_sequence
Variants Sequence variants in genomic regions with frequency and consequence info ensembl_get_variants
Homologs Orthologs and paralogs across multiple species ensembl_get_homologs
Regulation Enhancers, promoters, transcription factor binding sites ensembl_get_regulation
Coordinate Liftover Convert genomic coordinates between assembly versions liftover_coordinates

MCP Tools in Datasets MCP Server

Gene Lookup

ensembl_lookup_gene

Find genes by symbol or Ensembl ID.

Parameters: gene (string: ID or symbol), species (optional)

Returns: Gene ID, description, location, strand, exons, biotype

Sequence Retrieval

ensembl_get_sequence

Retrieve gene, CDS, or protein sequences.

Parameters: gene_id (string), type (optional: genomic, cds, protein)

Returns: FASTA sequence with metadata

Variant & Homolog Search

ensembl_get_variants

Find variants in a genomic region.

Parameters: region (string, e.g. "1:1000000-2000000"), species (optional)

Returns: Variant locations, alleles, consequences, frequencies

ensembl_get_homologs

Find orthologs in other species.

Parameters: gene_id (string), target_species (optional)

Returns: Orthologous genes with %identity, alignment information

Genomic Utilities

liftover_coordinates

Convert genomic coordinates between assembly versions (e.g., hg19 ↔ hg38).

Parameters: region (string), from_build (string), to_build (string)

Returns: Converted coordinates with mapping quality information

Example Queries

Lookup a gene and get its sequence

"Find the BRCA1 gene in the latest human assembly and get both the genomic and protein sequences."

Tools: ensembl_lookup_gene + ensembl_get_sequence

Find variants in a region

"What variants are found on chromosome 17 between positions 41196312 and 41277500 (BRCA1 region)?"

Tool: ensembl_get_variants

Identify orthologs

"Find orthologs of the human TP53 gene in mouse, zebrafish, and fruit fly."

Tool: ensembl_get_homologs

Coordinate conversion

"Convert the genomic coordinates chr1:1000000-2000000 from hg19 to hg38."

Tool: liftover_coordinates

Supported Species

  • Mouse (Mus musculus)
  • Rat (Rattus norvegicus)
  • Fruit fly (Drosophila melanogaster)
  • Yeast (Saccharomyces cerevisiae)
  • Nematode (Caenorhabditis elegans)
  • Zebrafish (Danio rerio)
  • Chicken (Gallus gallus)
  • Dog (Canis familiaris)
  • Primate species, plants, fungi, and more...
