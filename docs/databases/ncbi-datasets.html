NCBI Datasets

Programmatic Access to NCBI Reference Genomes, Gene Information, and Sequence Data

Home › Databases › NCBI Datasets

Overview

NCBI Datasets is the National Center for Biotechnology Information's programmatic interface for accessing genomic data. It provides rapid summaries of reference genomes, gene information, protein sequences, and assembly annotations. The server integrates with the NCBI Datasets command-line tool for efficient data retrieval and batch processing.

No API Keys Required — All NCBI Datasets tools are freely accessible. The MCP server uses the NCBI Datasets CLI (which can be installed optionally) or REST endpoints for data retrieval.

Key Features

  • Reference genome assembly information
  • Gene summaries and annotations
  • Batch gene queries
  • Sequence downloads in FASTA format
  • Taxon-based genome searches
  • GFF/GCF file information

Use Cases

  • Look up genome assembly versions
  • Fetch gene information across organisms
  • Batch retrieve sequences for multiple genes
  • Identify organisms by taxonomy
  • Download complete genome data
  • Access gene annotations and exon structures

Available Data Types

Data Type Description Accessible Via
Genome Summaries Assembly statistics, version information, organism details datasets_summary_genome
Gene Summaries Gene names, symbols, descriptions, locations, protein information datasets_summary_gene
Gene Downloads Full gene data packages with sequences and annotations datasets_download_gene
Genome Downloads Complete assembly sequences and annotation files datasets_download_genome
Batch Gene Queries Process multiple gene IDs/symbols in a single query batch_gene_summary
Protein Information Associated protein sequences and product information Gene summaries include protein details

MCP Tools in Datasets MCP Server

Genome Operations

datasets_summary_genome

Fetch assembly summaries for an organism or tax ID.

Parameters: organism (string), tax_id (integer)

Returns: Assembly name, version, organism info, contig/scaffold counts

datasets_download_genome

Download sequence data for a genome assembly.

Parameters: accession (string), format (optional: fasta, gff3)

Returns: Download URL, file information, data summary

Gene Operations

datasets_summary_gene

Retrieve gene information by symbol, ID, or genomic location.

Parameters: gene (string, ID or symbol), organism (optional), tax_id (optional)

Returns: Gene name, description, chromosomal location, exon structure, protein accession

datasets_download_gene

Download complete gene data packages.

Parameters: gene (string), organism (optional), include_protein (boolean)

Returns: Package location, contents summary, sequence information

batch_gene_summary

Process multiple genes in a single query with automatic deduplication.

Parameters: genes (comma-separated list or array), organism (optional)

Returns: Gene summaries for all queried genes

Example Queries

Get the latest human genome assembly

"What is the current human genome reference assembly version (GRCh38/hg38)?"

Tool: datasets_summary_genome with organism="Homo sapiens"

Look up a specific gene

"Give me the genomic location and exon structure of the human BRCA1 gene."

Tool: datasets_summary_gene with gene="BRCA1"

Batch query multiple genes

"Summarize TP53, EGFR, BRCA1, and PTEN for me — include chromosomal locations and protein sizes."

Tool: batch_gene_summary with genes=["TP53", "EGFR", "BRCA1", "PTEN"]

Find genes in a region

"What genes are located on chromosome 17 between positions 41000000 and 42000000?"

Tool: datasets_summary_gene with taxon-based search

Integration with Datasets MCP

NCBI Datasets tools work seamlessly with other tools in the MCP server:

  • With Ensembl: Retrieve gene summary from NCBI, then get detailed variant information from Ensembl
  • With UniProt: Get NCBI gene info, then fetch protein domain details from UniProt using the protein accession
  • With ClinVar: Identify genes from NCBI, then search for clinical variants in those genes
  • With PDB: Get protein info from NCBI gene data, then find 3D structures in PDB

Technical Details

Data Source

  • NCBI Reference Sequences (RefSeq)
  • GenBank assemblies
  • Official genome builds
  • Curated gene annotations

Implementation

  • Uses NCBI Datasets CLI (optional)
  • Fallback to REST API
  • Caching for frequent queries
  • Error handling for invalid inputs

Requirements

NCBI Datasets CLI (Optional) — Install for full functionality:

conda install -c conda-forge ncbi-datasets-cli
# or download from: https://ftp.ncbi.nlm.nih.gov/pub/datasets/command-line/

REST-only fallback available if CLI not installed.
